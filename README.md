# b-html-cli

A CLI for [b-html](https://github.com/b-html/b-html).

## Installation

```
$ npm install -g b-html-cli
```

## Usage

1\. `index.bhtml` -> `index.html`

```
$ b-html index.bhtml
```

2\. `bhtmls/**/*.bhtml` -> `htmls/**/*.html`

```
$ b-html -o htmls/ bhtmls/
```
